J Roth standard PCB format

I use this template for almost all PCBs that I design for the lab to try and standardize connectors and enclosures.

Features

  • 100mm eurocard format, fits inside Hammond 1457 series enclosures.Alternatively place inside a Schroff 24563131 Eurocard subrack.
  • S10B-ZR-3.4 10-pin JST ZH connector for power connections. Use with the JST A10ZR10ZR28H51A cable.
  • S4B-PH-K-S 4-pin connector for I2C connections. This is compatible with the Adafruit STEMMA standard. Use with a cable from Adafruit, such as the ST PH 2mm 4-Pin to Male Header Cable - I2C STEMMA Cable or STEMMA Cable - 4 Pin JST-PH 2mm Cable-Female/Female.
  • x6 Amphenol 132289 SMA connectors, remove if unnecessary.
